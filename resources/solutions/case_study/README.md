## Solutions of the main case study of NYC

`c1/` contains all solutions for runs with capacity 1 and variable pickup and delivery time (T/C), as well as including varying fleet size for the status quo, where buffer = 0 (`b0m`).

`c3-fp/` contains all solution for capacity 3 and fixed pickup time (RS/B).

`c3/` contains all solution for capacity 3 and variable pickup and delivery time (RS/C).
